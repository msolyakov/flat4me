Директория, содержащая компилированные сборки проектов солюшена Flat4me.Core.sln.

Служит источником для референсов на общие инфраструктуктурные библиотеки, такие как RabbitMQ.Client, Windows Azure Storage и Dapper.
Такое решение было принято из-за сбоев, свяpанных с тем, что разные проекты ссылались на разные версии библиотек из NuGet.

Теперь ссылки из NuGet на RabbitMQ.Client, Windows Azure Storage и Dapper остались только в Core и Data проектах.
Остальные (web, services) сслылаются на сборки в этой директории.
